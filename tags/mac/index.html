PYTHONPATH is currently: "/usr/local/lib/python3.9/site-packages:" You should `unset PYTHONPATH` to fix this. $ unset PYTHONPATH  で、この後pythonを実行すると2系になってしまう。 $ python WARNING: Python 2.7 is not recommended.
